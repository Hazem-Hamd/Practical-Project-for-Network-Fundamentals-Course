# Practical-Project-for-Network-Fundamentals-Course
 enterprise network infrastructure for a three-branch corporate environment using Cisco Packet Tracer.  Implemented advanced Layer 2 and Layer 3 protocols, including VLSM subnetting, VTP, EtherChannel, and RIPv2 routing.  Secured network access and remote management by deploying Port Security, SSH, and Extended Access Control Lists (ACLs).
